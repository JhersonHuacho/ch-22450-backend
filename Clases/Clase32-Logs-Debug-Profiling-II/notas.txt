* Artillery

- Podemos hacer pruebas de carga o pruebas de estres

$ node index.js 8081
$ node index.js 8081 CLUSTER

$ artillery quick --count 50 -n 40 http://localhost:3002?max=1000 > result_fork.txt

$ curl -X GET "http://localhost:3002/newUser?username=francisco&password=123456"

$ node --prof index.js
$ node --prof-process bloq-v8.log > result_prof-bloq.txt
